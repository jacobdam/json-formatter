# json-formatter

http://jacobdam.github.io/json-formatter makes JSON easier to read.

## Development

Install

```bash
git clone git@github.com:jacobdam/json-formatter.git
cd json-formatter
npm install
```

Test

```bash
npm run test
```

Publish to web

```bash
mkdir - build/web
git clone -b gh-pages git@github.com:jacobdam/json-formatter.git build/web
npm run publish-web
```
